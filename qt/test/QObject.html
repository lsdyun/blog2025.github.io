Qt 类继承关系可视化

QWidget 作为所有用户界面类的基类，继承自 QObject 和 QPaintDevice 类。本可视化展示了这些核心类的关系、功能及使用示例。

类继承关系图

QObject
Qt 对象模型的核心基类，提供信号槽机制、对象树管理、事件处理等核心功能。
QPaintDevice
所有可绘制对象的基类，定义绘图设备的抽象接口，如屏幕、打印机、图像缓冲区等。
QWidget
所有用户界面组件的基类，结合了 QObject 和 QPaintDevice 的功能，提供窗口系统集成、事件处理、布局管理和绘制能力。

类特性与功能

QObject 核心功能

  • 父子对象关系管理
  • 信号与槽通信机制
  • 对象属性系统
  • 事件处理框架
  • 定时器支持
  • 国际化支持

QPaintDevice 核心功能

  • 定义绘图设备接口
  • 管理设备坐标系统
  • 处理设备相关属性
  • 支持不同绘制后端
  • 提供设备度量信息

QWidget 核心功能

  • 窗口系统集成
  • 事件处理与传播
  • 布局管理系统
  • 样式与主题支持
  • 焦点与键盘处理
  • 国际化与可访问性
QWidget 使用示例 C++ 代码 
// 创建自定义窗口部件
class MyWidget : public QWidget {
    Q_OBJECT // 启用元对象系统功能

public:
    MyWidget(QWidget *parent = nullptr) : QWidget(parent) {
        // 创建子部件
        QLabel *label = new QLabel(tr("Hello, QWidget!"), this);
        QPushButton *button = new QPushButton(tr("Click me"), this);
        
        // 设置布局
        QVBoxLayout *layout = new QVBoxLayout(this);
        layout->addWidget(label);
        layout->addWidget(button);
        
        // 连接信号槽
        connect(button, &QPushButton::clicked, this, &MyWidget::onButtonClicked);
    }

private slots:
    void onButtonClicked() {
        // 处理按钮点击事件
        QMessageBox::information(this, tr("Message"), tr("Button clicked!"));
    }

protected:
    // 重写绘图事件
    void paintEvent(QPaintEvent *event) override {
        QPainter painter(this);
        painter.setRenderHint(QPainter::Antialiasing);
        
        // 绘制自定义背景
        painter.fillRect(rect(), QColor("#2c3e50"));
        
        QWidget::paintEvent(event);
    }
};
