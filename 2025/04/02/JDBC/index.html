一、什么是JDBC？

JDBC（Java Database Connectivity）是 Java 语言用于连接和操作关系型数据库的标准 API，提供了一套统一接口，支持与多种数据库交互‌。

二、模板

详解：

  • Driver 接口：‌定义驱动能力，需通过 Class.forName() 加载驱动类（如 com.mysql.jdbc.Driver）‌。
  • ‌**DriverManager 类**‌：管理驱动并创建数据库连接（Connection）‌。
  • ‌**Connection 接口**：‌代表与数据库的会话，用于创建 Statement 或 PreparedStatement‌。
  • ‌**Statement 与 PreparedStatement**：‌执行 SQL 语句，后者可预编译并防止 SQL 注入‌。
  • ‌**ResultSet 接口**：‌封装查询结果集，支持遍历和操作数据‌。

三、补充

  1. 事务管理：
    • ‌开启事务‌：conn.setAutoCommit(false);
    • ‌提交/回滚‌：conn.commit(); 或 conn.rollback();‌
  2. 防止 SQL 注入：
    • 使用 PreparedStatement 替代 Statement，预编译 SQL 并参数化输入‌。
  3. 连接泄漏：
    • ‌使用 try-with-resources 自动关闭资源‌。
  4. 编码问题：
    • ‌在 JDBC URL 中指定字符集（如 useUnicode=true&characterEncoding=UTF-8）‌
